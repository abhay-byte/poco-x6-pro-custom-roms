### How to Flash AOSP ROMs

```sh
# Flash recovery
fastboot flash vendor_boot_ab vendor_boot.img

# Reboot to recovery
fastboot reboot recovery

# Sideload ROM (from recovery, after selecting "Apply update via ADB")
adb sideload ROM.zip
```
